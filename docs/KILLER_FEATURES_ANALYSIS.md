### 1. SNR сети = кривизна V_Art
Для φ-взвешенной сети сенсоров: **SNR_φ = φ³ = V''(s₀)** — максимальное отношение сигнал/шум РАВНО кривизне потенциала Артосферы в вакууме.

### 2. Convergence Index
C_k = 1/(1 + φ·δ²), где δ = нормированное отклонение. При 1-sigma: **C(1σ) = 1/φ² = s₀** — совпадает с вакуумом V_Art.

### 3. Sybil-порог
Кластер из m Sybil-нод имеет влияние I(m) = m/φᵐ. Безопасно при m ≥ 7 (I < sin²θ₁₂ = 0.309).

### 4. ζ_DAO(s) — дзета-функция DAO
Спектральная тройка (C(Graph), l²(V), T_rep) порождает zeta_DAO(s) = Σλ_k^{-s}.

### 5. φ-гиперболическое дисконтирование
V_φ = Σ CF(nτ) · φ⁻ⁿ / F(n+2). Для постоянного потока: сумма = CF · √5 · φ.
